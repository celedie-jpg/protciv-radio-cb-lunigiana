ℹ️ Come aggiungere nuovi utenti
Per aggiungere nuovi utenti al sistema, modifica l'array utenti nel file login.html:
const utenti = [
  { username: 'admin', password: 'admin123', ruolo: 'admin', nome: 'Amministratore', volontarioId: null },
  { username: 'mario.rossi', password: 'volontario123', ruolo: 'volontario', nome: 'Mario Rossi', volontarioId: 1 }
];

Ruoli disponibili: admin (accesso completo) | volontario (solo propri servizi)
volontarioId: Collega l'utente a un volontario esistente (vedi tab Volontari per gli ID)
